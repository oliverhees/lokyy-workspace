# Lokyy Workspace — Documentation Site

Built with [Astro Starlight](https://starlight.astro.build/).

```bash
cd docs-site
npm install     # or: bun install
npm run dev     # local dev server
npm run build   # static build → ./dist
```

Content lives in `src/content/docs/` (German root locale + `en/` for English).
**Write docs alongside the task you implement** — every completed task updates the relevant page.
