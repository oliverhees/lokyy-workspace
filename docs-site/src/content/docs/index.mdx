## Was ist Lokyy?

Lokyy Workspace ist nicht nur ein weiteres KI-Chat-Tool, sondern ein **KI-Betriebssystem**: die zentrale Plattform, über die du deine gesamte KI-gestützte Arbeit betreibst — lokal oder auf deinem eigenen Server.
