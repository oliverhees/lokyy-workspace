## Voraussetzungen (geplant)

- Docker & Docker Compose
- (optional) ein lokales LLM via Ollama/vLLM/llama.cpp

## Schnellstart (geplant)

```bash
git clone https://github.com/oliverhees/lokyy-workspace.git
cd lokyy-workspace
cp .env.example .env
docker compose up -d --build
```
